How do I find what starting index to use?

Open the world folder for your world. You can open it using the world select menu: Select your world, click "Edit" at the bottom, and then click "Open World Folder."

In 26.1 and above, your maps are in the data/minecraft/maps folder (create the folder if it doesn't exist yet). If there are no files named with numbers, use 0 as your starting index, otherwise add 1 to the biggest numbered file and use that as your starting index.

In 1.21.11 and below, the maps go directly in the data folder. If there are no files starting with "map_" use 0 as your starting index. Otherwise, add 1 to the biggest numbered file prefixed with "map_" and use that as your starting index.

Where do I extract the map files?

You will need access to the world files. Before doing anything, make sure you exit the world (or shut off the server if you're playing on a server) to make sure everything saves properly.

Find the folder in which your maps are stored in as shown above. Then, extract the map files into this folder. Make sure you replace last_id.dat (26.1+) or idcounts.dat (1.21.11 and below)! You can re-enter your world once you're done.

Where do I put the datapack and how do I enable it?

The datapack goes into the datapacks folder in your world folder. You can be in the world while you do this. You do NOT need to extract the zip file.

After that, run the command /datapack list and then use /datapack enable on your datapack file to enable it (if you ran /datapack list the file should appear as an autocomplete suggestion).

Why is the animated map flickering and how do I fix it?

It does this because the game reads the map data from disk whenever the maps change, which is very slow. Eventually it gets better if the game decides to cache the map data so it doesn't have to read from disk every time.

You can stop the flickering by forcing the game to cache all the maps by running the cache command, which cycles through the maps using your offhand slot at around 20 maps per second. Note that if you have an item in your offhand when you run this command you will lose it! If there are a lot of maps or frames you can speed this process up in 1.20.3+ using the command /tick rate 1000 (note that this will speed up the rest of your game as well). Use /tick rate 20 to restore the default tick rate.

Why are animated map arts so loud?

The reason is that item frames always play a popping sound whenever the item inside changes. To turn this off, go to your game options, select Music & Sounds, and slide Friendly Creatures all the way to OFF.

I want to delete a datapack generated by this tool. How do I clean up the mess of scoreboard objectives created by it?

Copy and run the clean command. This will delete all scoreboard objectives created by the datapack. Then, use /datapack disable to disable the datapack and remove the datapack from your datapacks folder.

Does this work on survival worlds?

Running the commands used to obtain the item frames and map items, enable the datapack, etc. requires cheats or operator permissions in vanilla Minecraft. If you want to temporarily enable cheats to run the commands to set things up, press "Open to LAN", set "Allow Commands" to ON, and start the LAN world.
